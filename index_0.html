DataLoader
Unity - C# + C++ + Bases de datos
Unity - C# + C++ + Databases
DataLoader es una herramienta para Unity y, en un futuro, Unreal. Que guarda configuraciones tanto en una base de datos local como en un xml, a elección del usuario. Estas configuraciones se pueden establecer en cualquier actor que tenga el recipiente adecuado. En este caso, el recipiente es una variable del tipo DatabaseRow.
DataLoader is a tool for Unity and, in the future, Unreal. It saves configurations in a local database as well as in an xml, at the user's choice. These configurations can be set on any actor that has the right container. In this case, the container is a variable of the DatabaseRow type.
